EVM Watchtower

钱包登录、地址监控、三条 EVM 主网、Telegram 实时通知。

GitHub Features

Wallet-gated

用户必须使用 EVM 钱包签名登录，每个用户管理自己的监控地址和 Telegram 配置。

Three chains

先支持 Ethereum、Base、BNB Smart Chain，后续可以继续扩展 Arbitrum、Polygon 等网络。

Action parsing

解析收发、买卖、swap、授权、NFT/1155、合约调用和事件命中，并给出交易浏览器链接。

Telegram alerts

被监控地址一旦有新区块活动，就通过用户绑定的 Telegram 机器人通知。
